Le Document object Model (DOM)

Le Document Object Model est une représentation structurée d'une page web qui permet aux programmes informatiques de manipuler le contenu, la structure et le style d'une page web de manière dynamique. En d'autres termes, le DOM est un modèle de programmation qui représente une page web comme une hierarchie d'objets.
