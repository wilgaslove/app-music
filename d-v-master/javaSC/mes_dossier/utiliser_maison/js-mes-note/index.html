Programmation Objet Orienté.

Programmation Objet Orienté (POO).

La POO est un paradigme de Programmation qui permet d'organiser et de structurer le code en utilisant des concepts appelés objets

Les classes représentent la moule à utiliser pour créer des objets.

Un objet représente un concept, une idée ou toute entitée du monde physique, comme une voiture, une personne ou une page de livre.

Un paradigme de Programmation est une approche ou une méthode générale qui guide la façon dont on structure et organise le code d'un programme.

Les fonction dans un objet sont appelées méthode

Les variables sont appelées des propriétés auquelles on peut assigner des valeurs.
